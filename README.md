# License Plate Recognition System

A computer vision pipeline designed to recognize Malaysian license plates using OpenCV and Tesseract OCR without relying on heavy deep-learning frameworks.

## Features
* Image preprocessing using CLAHE, Grayscale conversion, and Sobel edge detection.
* Structural regex matching to correct distorted character extractions.
* Optimized morphological transformations for clear contour mapping.
